# 📦 InvenTrack

InvenTrack is a **Flutter-based inventory management mobile application** designed to help users track items, manage quantities, and maintain inventory efficiently. The project is developed as a **personal open-source application**, with complete ownership of design and implementation.

---

## ✨ Features

- 📋 Inventory listing with item details  
- ➕ Add, update, and manage inventory items  
- 🔄 Quantity and status tracking  
- 📱 Clean, responsive, and user-friendly UI  
- 🧩 Modular and reusable Flutter widgets  

---

## 🛠️ Tech Stack

- Flutter  
- Dart  
- Git & GitHub
